In this assignment you will practice in implementing mean encodings using final project dataset. You are asked to encode `item_id` in 4 different ways:

1) Via KFold scheme with number of folds = 5. Use KFold(5) from sklearn.model_selection;

2) Via Leave-one-out scheme;

3) Via smoothing with alpha = 100. Use the formula from the first slide in the Regularization video and 0.3343 as globalmean;

4) Via expanding mean scheme.

Fill NANs in the encoding with 0.3343.

The answer will be the correlation coefficient between resulting encoding and target variable up to 4 decimal places.

Some encoding schemes depend on sorting order, so in order to avoid confusion, please use the provided notebook .
